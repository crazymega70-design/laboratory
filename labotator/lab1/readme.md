1. Чем отличается var от let и const? Область видимости: var виден во всей функции, а let и const — только внутри ближайших фигурных скобок {}. Переназначение: let и var позволяет менять значение переменной, а const — нет так как это константа. Переменные var их можно использовать до объявления (будет undefined). let и const так не умеют.
2. Что такое неявное преобразование типов? Это когда JavaScript сам меняет тип данных, чтобы попытаться выполнить операцию. Пример с плюсом: "5" + 2 превратится в "52" (число станет строкой). Пример с минусом: "5" - 2 превратится в 3 (строка станет числом).
3. Как работает оператор == в сравнении с ===? == (нестрогое равенство): Сравнивает только значения, предварительно пытаясь привести их к одному типу. Поэтому 5 == "5" вернет true. === (строгое равенство): Сравнивает и значение, и тип. Если типы разные, он сразу вернет false. Поэтому 5 === "5" вернет false.
