Первая лабараторная работа по JS 
Описание лабораторной работы
Данная работа посвящена изучению основ языка программирования JavaScript. В ходе проделанной работы были освоены навыки запуска кода в консоли браузера, создание HTML-страниц с внедренным и внешним JS-кодом, а также работа с базовыми типами данных, логическими условиями и циклами. 2. Инструкции по запуску проекта

Для запуска проекта нужно:

Создать файлы index.html и script.js. Поместить их в одну папку. Открыть файл index.html. Для просмотра результатов работы кода: Нажать F12, чтобы открыть инструменты разработчика. Перейдите на вкладку Console.

Краткая документация к проекту
Проект состоит из двух основных файлов: index.html — HTML-страница, к которой подключен скрипт. script.js — файл с JavaScript кодом.
